# Puntual

Aplicación de escritorio para Linux con:

- bandeja del sistema
- persistencia local
- recuperación de alarmas tras reinicio
- cuenta atrás persistente
- empaquetado `.deb`

## Descargar

Descarga la última versión desde:

- https://github.com/jjdeharo/puntual/releases/latest

Instalación en Debian, Ubuntu, MX Linux y derivadas:

```bash
sudo apt install ./puntual_0.1.0_amd64.deb
```

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run dist
```
